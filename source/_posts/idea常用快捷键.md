总结一下Intellij IDEA常用到的一些还没记住的快捷键：

| 快捷键         | 作用                      |
| -------------- | ------------------------- |
| ctrl + alt + o | 删除类中没有用到的package |
| ctrl + F12     | 弹出当前类中的方法集合    |
| ctrl + j       | 可以自定义一些代码模板    |
