## 补码：

java程序中的运算都是补码运算

## 位运算符和移位运算

| 位运算符  | 运算规则                       | 举例   |
| --------- | ------------------------------ | ------ |
| 按位与&   | 两位全为1，结果为1             | 6&3=2  |
| 按位或\|  | 两位有一个1，结果为1           | 6\|3=7 |
| 按位异或^ | 两个一个为0，一个为1， 结果为1 | 6^3=5  |
| 按位取反~ | 0->1, 1->0, 符号位也要变       | ~6=-7  |
| <<        | 左移                           |        |
| ```>>```  | 右移                           |        |
| ```>>>``` | 无符号右移                     |        |

<!--more--> 

## 1. <<

<< 表示左移，不分正负数，低位补0

**e.g**：1 << 4 

​		0000 0001 左移4位  => 0001 0000  转换为十进制后等于 16
